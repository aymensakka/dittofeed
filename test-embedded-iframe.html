How to Use This Test Page

Step 1: Create a Child Workspace (if needed)

First, ensure you have a child workspace. You can create one using the Parent Admin API:

curl -X PUT http://localhost:3001/api-l/admin/workspaces/child \ -H "Authorization: Bearer YOUR_PARENT_ADMIN_KEY" \ -H "Content-Type: application/json" \ -d '{"name": "Test Child Workspace", "externalId": "test-123"}'

Step 2: Get a Session Token

Use your workspace's write key to create a session token:

curl -X POST http://localhost:3001/api-l/sessions \ -H "Authorization: Bearer YOUR_WRITE_KEY" \ -H "Content-Type: application/json" \ -d '{"workspaceId": "YOUR_WORKSPACE_ID"}'

Step 3: Enter Details Below

Enter your workspace ID and session token, then select a component to embed.
